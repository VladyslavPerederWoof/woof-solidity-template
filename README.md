# Woof Solidity Template

This repository is a starter template for Solidity projects. It ships with an opinionated, preconfigured toolchain so that new projects can be spun up quickly with consistent conventions across the codebase.

Out of the box it includes:

- **Hardhat 3** with the Mocha + Ethers toolbox for building and testing.
- **TypeScript** with ESLint, Prettier, and `eslint-config-prettier` preconfigured.
- **Solidity linting & formatting** via Solhint and `prettier-plugin-solidity`.
- **Hardhat Ignition** for reproducible deployments to mainnet and testnets.
- **NatSpec-based contract docs** generated by `@solarity/hardhat-markup`.
- **ABI export** via `@solidstate/hardhat-abi-exporter` and contract size reporting.
- **Security tooling** scripts for Slither and Aderyn.
- **Git hygiene** with Husky, Commitlint, and conventional commits.

> The sections below (**Description**, **Architecture**, **Roles**, **Flows**, **Configuration Setters**, **Deployment → Configuration**, etc.) are intentionally left as placeholders. They are designed to be edited per project, so when you bootstrap a new repo from this template, fill them in with the specifics of your contracts, actors, and flows.

Contract documentation is auto-generated from NatSpec comments:

```bash

```

Generated markdown is written to `docs/`.
